# 🎓 CampusConnect

CampusConnect is a Student Community Platform built to help students connect, share resources, and stay updated with college activities.

## 🚀 Features

- 📚 Notes Sharing
- 🎉 College Events
- 🛒 Student Marketplace
- 🔍 Lost & Found
- 🤖 AI Assistant
- 👤 User Login & Signup
- 📌 Student Dashboard

## 🛠️ Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- JSON Storage

## 💻 Installation

Clone repository:

```bash
git clone https://github.com/hastisangani18/CampusConnect.git
```

Install packages:

```bash
npm install
```

Run backend:

```bash
node server.js
```
